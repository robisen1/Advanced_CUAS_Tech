RF Jamming
Techniques
Taxonomy

RF-based jamming remains one of the least understood disciplines in C-UAS practice — yet it is precisely where the gap between standard EW curricula and operational reality is widest. Knowing these techniques is what separates reactive operators from effective ones. Tap any tile to expand.

01 Temporal / Duty-Cycle Manipulation
Pulsed Jamming
Cyclic on/off defeats AGC and conserves transmitter power

02 Protocol / Behavioral Exploitation

ACK Starvation

Hop Prediction Jamming

03 Directed / Spatial Techniques
Null-Steering Jamming
Phased array beams on target while nulling toward friendlies
→
Follow-Me Tracking
Beam steered continuously to moving target via AoA / Doppler

Link-Selective Jamming

04 Signal Design Techniques
Swept / Chirp Jamming
Continuous freq sweep covers all FHSS hops over time
→
Spot + Barrage Combination

05 Effect-Shaping — Induced Behavior
RTH Induction
Jam control link only — drone autonomously returns to home point
→
Hover-Lock
Jam control link on hover-hold platforms — drone freezes for ID
→
Geofence Spoofing

06 Covert / Low-Probability-of-Intercept
Noise-Floor Creep

→
Fractional Bandwidth Jamming
Target only pilot/sync subcarriers in OFDM — receiver loses sync quietly
